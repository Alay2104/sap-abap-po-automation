
# Automated Purchase Order Management System (SAP ABAP)

This project automates SAP Purchase Order creation using ABAP.

## Features
- Custom Purchase Request Entry
- Auto Purchase Order Creation via BAPI
- Smart Form PO Print
- ALV Dashboard for Tracking

## SAP Objects Used
- Data Dictionary (SE11)
- ABAP Reports (SE38)
- BAPI_PO_CREATE1
- Smart Forms
- ALV Grid

## How It Works
1. User enters purchase request
2. System saves request in custom table
3. PO generated automatically using BAPI
4. PO printed using Smart Form
5. Dashboard shows all POs
